### 2. 安装文泉艺字体、samba、wodim、expect、vim、build－essential、cmake
### 3. 安装solarized的ls配色
### 4. 安装vim插件管理vundle并配置插件
### 5. 安装Tmux并配置插件
### 6. 安装google chrome浏览器、搜狗拼音、GIMP矢量图、[BaiduPCS]及迅雷（linux）
